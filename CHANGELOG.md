# Changelog

All notable changes to README.ai will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.2.0] - 2026-02-11

### Added
- Searchable profile card directory with live filtering (GitHub Pages)
- GitHub Pages support with Jekyll (`_config.yml`, pretty permalinks)
- GitHub Action to auto-rebuild directory index on profile changes
- SECURITY.md with privacy guidance
- CHANGELOG.md, CODE_OF_CONDUCT.md, ROADMAP.md
- GitHub avatar support in profile cards
- Skip buttons for optional fields (no more typing "skip")

### Changed
- Rebranded from MeetMe to README.ai
- Expanded profile to 17 questions (added start date, GitHub handle, social handles)
- Profile "About" section now prominently shows name, role, team
- Social handles split into individual rows per platform

### Fixed
- Remaining "MeetMe" branding in templates, profiles, and LICENSE
- Question count inconsistency (14 vs 17)
- Placeholder "yourorg" URLs replaced with actual repo URLs
- Jekyll permalink configuration for proper page linking
- Removed unnecessary `bash` from agent tool list

## [0.1.0] - 2026-02-11

### Added
- Initial release as MeetMe
- Copilot CLI agent with 14-question guided profile builder
- 3 example profiles (Jane Doe, Marcus Chen, Sofia Rivera)
- Skill file with writing guide and tone mapping
- Profile template with Markdown formatting
- ONBOARDING.md for first-time CLI users
- README, LICENSE, CONTRIBUTING
